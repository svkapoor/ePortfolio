Discovery Project: Arduino Motion Detection Security System

This project uses an Arduino and a PIR sensor to detect motion and simulate a simple security system. When motion is detected, the system can trigger a notification mechanism such as a buzzer, an LED indicator, or a message-based alert.

Overview

Build a basic security system prototype that senses movement and responds with an alert.

Role

Design the setup, connect the Arduino and PIR sensor, program the logic, and test the notification behavior.

Skills

Arduino programming, sensor integration, circuit setup, debugging, and basic system design.

Outcome

A working motion-detection prototype that demonstrates how a simple embedded system can monitor activity and respond in real time.

Why it matters

This project matters because it connects hardware and software in a practical way. It also reflects my interest in security-related systems and gives me experience building a small but realistic prototype from components, code, and testing.

Secure Authentication System + Zero-Knowledge Vault

Overview

A full-stack authentication system with a built-in password manager designed so sensitive user data is never exposed to the server.

Role

I developed the backend authentication flow, designed the vault security model, and implemented secure session and token handling.

Skills

Flask, Argon2, JWTs, CSRF protection, PBKDF2, AES-256-GCM, session security, Docker, PostgreSQL, and production deployment.

Outcome

The system supports secure authentication, encrypted credential storage, client-side vault encryption, and browser-based vault access through a Chrome extension.

Why it matters

This project matters because it demonstrates practical security engineering, including zero-knowledge design, secure authentication, and production-ready system architecture.

Real-Time Deepfake Detection System

Overview

A machine learning system that detects deepfake faces in live video streams and supports real-time monitoring use cases.

Role

I built the model pipeline, trained the system on large-scale face data, and designed the real-time inference workflow for live analysis.

Skills

PyTorch, ResNet-18, computer vision, model training, real-time inference, API integration, and evaluation on large datasets.

Outcome

The model was trained on more than 150,000 face crops, achieved about 88% validation accuracy, and supported a live detection pipeline with Zoom session flagging.

Why it matters

This project matters because it applies machine learning to a real-time security problem and focuses on deployment in live environments instead of only offline accuracy.
